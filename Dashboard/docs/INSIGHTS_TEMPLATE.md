## Caveats / limitations
- Stock universe is a representative basket (~25 stocks across 8 sectors),
  not the full Nifty 50 — results are directional, not exhaustive.
- Returns are price-return only (dividends not included), so total return
  figures slightly understate actual investor returns.
- Daily close-based returns; intraday volatility is not captured.
